## 🎖️ 项目简介

**HBTI (Helldiver Personality Index)** 是一项由超级地球真理部授权开发的心理评估工具。在每一位新兵踏入空降仓、成为伟大的绝地潜兵之前，必须通过此系统来确定其在民主战场上的潜在定位。

通过 20 道严苛的心理测试题目，系统将分析你的忠诚度、决策效率以及对 710 元素的渴望程度，最终将你匹配至 16 种预设人格之一（包括各种敌方单位、昂贵的 500KG 炸弹，以及极少数真正的绝地潜兵）。

## 🚀 系统特性

- **沉浸式 CRT UI**: 完美还原超级地球终端机视觉效果，包含扫描线、闪烁特效和真理部滚动标语。
- **16 种深度人格**: 涵盖虫族、机器人、光能族以及传说中的“绝地潜兵”人格。
- **权重评分算法**: 每一道题的选项都经过精密计算，确保评估结果符合管理民主的统计学规律。
- **海报生成功能**: 使用 `html2canvas` 技术，一键生成属于你的新兵档案，方便在社交媒体传播。
- **零依赖部署**: 纯 HTML/JS/CSS 实现，通过 CDN 加载库文件，支持拖拽至 Cloudflare Pages 立即上线。

## 🛠️ 部署指南

本系统设计极其符合行政效率，无需任何 Node.js 环境：

1. 克隆本仓库或下载 `hbti` 文件夹。
2. 将整个文件夹拖入 **Cloudflare Pages** 的上传区域。
3. 立即向全人类分享你的民主身份！
